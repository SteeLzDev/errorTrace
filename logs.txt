Migração completa do provedor de SMS de Twilio para Zenvia API. A implementação:

Utiliza RestTemplate para chamadas HTTP à API REST da Zenvia (POST /v2/channels/sms/messages)
Centraliza configurações via SysConfig (@Value), compatível com application.properties (dev) e KeyVault Docker (produção)
Adiciona suporte a caracteres especiais via encodingStrategy: MORE_CHARACTER_SUPPORT
Registra cada envio de SMS em tb_ocorrencia_usuario (TOC 266) com dados do destinatário e mensagem
Implementa controle de cotas mensais por consignante via tb_dados_consignante (TDA 115/116/117/118)
Verifica limites inferior (TPC 1016) e superior (TPC 1017) antes de cada envio
Envia alertas automáticos por e-mail quando limites são atingidos
Segue princípios SOLID: SMSHelper (envio + controle), SmsRequest (DTO), SysConfig (configuração), EnviaSMSHelper (orquestração)
